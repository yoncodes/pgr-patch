## Pgr Patch

## How to use
```bash
Cargo build --release
```

- Download and use the shim method [here](https://github.com/Xpl0itR/VersionShim)
- add lucia.dll to libary.txt
- Place lucia.dll in the same directory as PGR.exe
- KRSDK.dll in PGR_Data\Plugins (make sure to save the old one to .bak or something)

- run the game
